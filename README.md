## Install

```.bash
# install from github
git clone https://github.com/Bit03/django-spectre.git
cd django-spectre
python setup.py install
```


## Configuration
Add to INSTALL_APPS in your settings.py
```.python
INSTALLED_APPS = [
    ...
    'spectre',
]


# spectre config
#
# -------------------------------------------------
SPECTRE_DEFAULTS = {
    "css_url": {
        "url": "https://unpkg.com/spectre.css/dist/spectre.min.css"
    },
}

```
