# django-spectre

## Overview 

Spectre css Docs

https://picturepan2.github.io/spectre/index.html


## Install

```.bash
# install from github
git clone https://github.com/Bit03/django-spectre.git
cd django-spectre
python setup.py install
```


## Usage
add spectre package into INSTALL_APPS in settings.py
```.python
INSTALLED_APPS = [
    ...
    'spectre',
]
```
